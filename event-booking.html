Event Booking System

Java (OOP) • MySQL • Admin Dashboard

About the Project

The Event Booking System is a Java application built using object-oriented programming principles, backed by a relational MySQL database. It manages customers, appointments, and services, giving users a straightforward way to book and track events. The system also includes an admin dashboard that allows an administrator to confirm, update, and cancel bookings, keeping the schedule accurate and up to date.

The project focuses on clean data modelling: customers, services, and appointments are each represented as their own classes and connected through relational keys in the database, making it easy to extend the system with new features such as reporting or notifications.
